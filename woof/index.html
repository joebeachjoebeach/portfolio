WoofJS Projects

As a coding teacher at The Coding Space, I've had a number of students using WoofJS as a learning tool. It makes the transition from block-based coding, like Scratch, to text-based coding less intimidating and more fun.

These are projects that I've made using WoofJS; some were made to prepare myself to teach with WoofJS, and others were made as contributions to the WoofJS tutorial collection. In the latter case, I've included a link to the tutorial I made.
